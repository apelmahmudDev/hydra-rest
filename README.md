## Overview

HydraRest helps you track two core recovery habits in one place:

- Water intake with quick-add controls and a custom amount input
- Sleep tracking with goal comparison and status feedback
- Daily progress rings and stats for fast at-a-glance updates
- Weekly summaries and calendar history for longer-term insight

## Features

- Responsive single-page UI
- Light and dark themes
- Animated progress rings for water and sleep
- Editable daily goals
- Calendar and day-detail views
- Local storage persistence for your tracked data

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser, or serve the folder with a local web server.
3. Start tracking your water and sleep habits.

## Project Structure

- `index.html` - App shell and UI markup
- `style.css` - Main stylesheet entry point
- `src/css/` - Modular CSS layers for layout, cards, calendar, and more
- `src/js/` - Application logic, state, storage, and UI updates
- `assets/` - Logo and card background SVGs

## Source

The app is built as a small static front end, so the source is easy to inspect and extend. The main entry point is [index.html](./index.html), and the visual assets live in [assets](./assets).
